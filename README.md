# Управление рецептами

Консольное приложение для управления рецептами, масштабирования порций и генерации списков покупок.

## Установка

git clone https://github.com/EMiN-nn/HW2tpp.git \
cd HW2tpp \
pip install -r requirements.txt

## Использование

Запуск тестов осуществляется командой:
pytest
